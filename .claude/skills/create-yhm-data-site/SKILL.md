Use `site-cli` (`target/release/site-cli`) to scaffold site structure (pages, collections, entries), then write content manually. Build with `cargo build --package site-cli --release` if binary is missing.

See [workflow.md](workflow.md) for the creation process — requirements, site-cli commands, content creation, and validation.
See [reference.md](reference.md) for all available elements, sitedef schema, image paths, and collection types.
See [examples.md](examples.md) for complete worked examples.
