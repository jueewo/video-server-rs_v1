API Keys vs Access Codes

API Keys are for scripts and automation (delete script, CLI, MCP server, WebDAV). Access Codes are for sharing resources with external viewers.

Replace YOUR_API_KEY with your actual key (shown once at creation time).

curl (HTTP API)

curl -H "Authorization: Bearer YOUR_API_KEY" \
     http://localhost:3000/api/media

WebDAV (macOS Finder / any client)

# Finder: Go → Connect to Server (⌘K)
# URL:      http://localhost:3001/dav/{workspace_id}
# Username: anything
# Password: YOUR_API_KEY

# curl
curl --user "username:YOUR_API_KEY" \
     -X PROPFIND http://localhost:3001/dav/{workspace_id}

Environment variable

export MEDIA_API_KEY="YOUR_API_KEY"

curl -H "Authorization: Bearer $MEDIA_API_KEY" \
     http://localhost:3000/api/media
