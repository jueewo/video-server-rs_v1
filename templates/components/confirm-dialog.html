{# Confirm Dialog Component (Alpine.js) Requires the parent element (or an ancestor) to have x-data="confirmDialog()". The confirmDialog Alpine component is registered globally in base-tailwind.html. Usage: 1. Add x-data="confirmDialog()" to a wrapper div around your list 2. Include this partial inside that wrapper: {% include "components/confirm-dialog.html" %} 3. Trigger from any button: NOTE: modal-open is bound dynamically via Alpine (:class) so that DaisyUI's :root:has(.modal.modal-open) scroll-lock rule only fires when the dialog is actually open. A static 'modal-open' class would be present in the DOM on page load and permanently suppress page scrolling via the CSS :has() selector. #}
